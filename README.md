Virtual Physics Simulation Lab
A comprehensive Python-based interactive simulation platform for 10 fundamental engineering physics experiments with real-time visualizations and data analysis.

 Running Individual Experiments
- Select experiment number (1-10) from the menu
- Follow the prompts to enter experimental parameters
- Input measurement data when requested
- View the generated plot and analysis results

 Running All Experiments
- Select option 11 to run all experiments automatically
- The program will generate plots for each experiment sequentially
- Close each plot window to proceed to the next experiment
